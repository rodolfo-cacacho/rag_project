import faiss
import numpy as np
from transformers import BertTokenizer, BertModel

tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertModel.from_pretrained('bert-base-uncased')

documents = [
    "What is the capital of France?",
    "How to bake a cake?",

    # Add other documents
]

encodings = [tokenizer.encode(doc, return_tensors='pt') for doc in documents]
with torch.no_grad():
    embeddings = np.array([model(enc)[0][:, 0, :].numpy() for enc in encodings])

### Step 4: Implement the Retrieval Mechanism

When a user inputs a query, retrieve the most relevant documents.

```python
def retrieve_top_k(query, k=5):
    query_encoding = tokenizer.encode(query, return_tensors='pt')
    with torch.no_grad():
        query_embedding = model(query_encoding)[0][:, 0, :].numpy()
    D, I = index.search(query_embedding, k)
    return [documents[i] for i in I[0]]
```

### Step 5: Combine with a Generative Model

Use a generation-based model like GPT-3 or T5 to generate responses based on the retrieved documents.

```python
from transformers import T5ForConditionalGeneration, T5Tokenizer

# Load pre-trained T5 model and tokenizer
t5_tokenizer = T5Tokenizer.from_pretrained('t5-small')
t5_model = T5ForConditionalGeneration.from_pretrained('t5-small')

def generate_response(query):
    retrieved_docs = retrieve_top_k(query)
    context = " ".join(retrieved_docs) + " " + query
    input_ids = t5_tokenizer.encode("summarize: " + context, return_tensors="pt", max_length=512, truncation=True)
    
    with torch.no_grad():
        outputs = t5_model.generate(input_ids, max_length=100, num_beams=5, early_stopping=True)
        
    return t5_tokenizer.decode(outputs[0], skip_special_tokens=True)
```

from pathlib import Path

    elements = partition_pdf(filename=file_path)
    content = "\n".join([element.text for element in elements if hasattr(element, 'text')])

def store_embeddings(embeddings, index_file="index.faiss"):
    embeddings_array = np.array(embeddings).reshape(1, -1)

[2024-06-05 14:38:26.016921] assistant: Effizienzhäuser, auch als KfW-Effizienzhäuser bekannt, sind Gebäude, die einen bestimmten energetischen Standard erfüllen, welcher durch die Kreditanstalt für Wiederaufbau (KfW) in Deutschland festgelegt wird. Diese Standards berücksichtigen den Energieverbrauch für Heizung, Warmwasser, Lüftung und Kühlung und sind darauf ausgelegt, den Energiebedarf eines Gebäudes zu minimieren.

Die Effizienzhaus-Standards werden durch Zahlenwerte wie z.B. Effizienzhaus 55, 70, 85 oder 100 bezeichnet. Diese Zahlenwerte geben an, wie hoch der Energiebedarf des Hauses im Vergleich zu einem Referenzgebäude gemäß der Energieeinsparverordnung (EnEV) ist:

- **Effizienzhaus 55** bedeutet, dass das Haus nur 55% der Energie eines Referenzgebäudes benötigt.
- **Effizienzhaus 70** liegt bei 70% des Energiebedarfs eines Referenzgebäudes usw.

Je niedriger die Zahl, desto energieeffizienter ist das Gebäude. Niedrigenergiehäuser oder Passivhäuser sind oft die ambitioniertesten Varianten und zielen darauf ab, den Energieverbrauch auf ein Minimum zu reduzieren bzw. auch Energiegewinne zu ermöglichen.

Der Bau oder die Sanierung zu einem Effizienzhaus kann durch die KfW gefördert werden, indem zinsgünstige Kredite oder Zuschüsse angeboten werden. Beispielsweise gibt es Programme wie "KfW-Programm 153 - Energieeffizient Bauen" oder "KfW-Programm 151/152 - Energieeffizient Sanieren", die Bauherren und Eigentümern finanziell entgegenkommen.

Diese Effizienzhäuser tragen nicht nur zur Reduktion des CO2-Ausstoßes und zum Klimaschutz bei, sondern sie bieten auch Vorteile wie geringere Energiekosten und ein verbessertes Wohnklima.
